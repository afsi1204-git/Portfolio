# Afsheen Fathima — Portfolio

A modern, animated portfolio website built with **React**, **Vite**, and **Framer Motion**, based on your resume.

## Features

- Animated hero with rotating roles and floating orbs
- Glassmorphism cards with hover transitions
- Scroll progress bar and section-aware navigation
- Animated skill bars, project cards, and certification list
- Contact form (opens your email client)
- Fully responsive mobile layout

## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or newer recommended)
- npm (comes with Node.js)

## How to Run

### 1. Open the project folder

```powershell
cd "C:\Users\afsi1\OneDrive\Desktop\Portfolio"
```

### 2. Install dependencies (first time only)

```powershell
npm install
```

### 3. Start the development server

```powershell
npm run dev
```

### 4. Open in your browser

Vite will print a local URL, usually:

```
http://localhost:5173
```

Open that link in Chrome, Edge, or Firefox.

## Other Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start local dev server with hot reload |
| `npm run build` | Create production build in `dist/` folder |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint checks |

## Deploy (Optional)

After `npm run build`, upload the `dist` folder to:

- [Vercel](https://vercel.com)
- [Netlify](https://netlify.com)
- [GitHub Pages](https://pages.github.com)

## Customize

Edit `src/data/portfolio.js` to update your info, projects, skills, and contact details without touching components.
